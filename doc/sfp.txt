 sfp.txt

@PURPOSE: Package with functional programming utilities.

@MNEMONIC: S functional programming (sfp)

@DESCRIPTION: This package contains a collection of commonly useful components
 related to functional programming.

/Component Synopsis
/------------------
: 'sfp_cf':
:      Provide routines that make curried versions of functions. 
: 'sfp_cfunc':
:      Provide metafunction for generating curried Boost.Function types.
: 'sfp_ciof':
:      Provide routines to curry IO functions.
: 'sfp_ciofunc':
:      Provide metafunction for generating curried IO Boost.Function types.
: 'sfp_flipped':
:      Provide a two-argument function wrapper that flips its arguments.
: 'sfp_fliputil':
:      Provide utilities for argument flipping of two argument functions.
